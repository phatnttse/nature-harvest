Upload (Upload Widget)

This example shows how to integrate the Cloudinary Upload Widget into an Angular application.

Please note that the following defaults are being used:
  • The upload source is limited to the local filesystem or to a remote URL.
  • The myphotoalbum-angular tag gets added to all photos uploaded via this method (this is used in the photo album to retrieve images).
  • Only images can be uploaded.
